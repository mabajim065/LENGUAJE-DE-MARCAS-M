Tipos de datos en JavaScript

  • Números: Como 5, 3.14, etc.
  • Cadenas: Texto, como "Hola" o "Juan".
  • Booleanos: Solo tienen dos valores, true (verdadero) o false (falso).
  • undefined: Cuando una variable no tiene valor porque no se ha creado.
  • null: Para decir que algo está vacío intencionalmente.

"use strict" y por qué usarlo

Antes, JavaScript permitía cosas como usar variables sin definirlas, lo que causaba errores raros. Si pones "use strict" al inicio de tu código, el lenguaje se vuelve más estricto.

"use strict";
let nombre = "Carmen"; // Bien
apellido = "Diaz"; // Error, porque no usé "let"

Constantes y variables

"use strict";
const gravedad = 9.81; // No cambia
let posicion = 42; // Puede cambiar

Cómo unir textos (concatenación)

Se puede hacer con + o con plantillas de ES6.

let nombre = "Carmen";
let apellido = "Diaz";
let completo = `${nombre} ${apellido}`; // "Carmen Diaz"

Poner JavaScript en HTML

Para usar JavaScript en una página web, se coloca dentro de <script>.

"use strict";
let numero = 42;
document.write(numero); // Escribe "42" en la página

Decisiones con "if"

"use strict";
let numero1 = 10;
let numero2 = 5;
if (numero1 > numero2) {
    document.write("El primero es mayor");
} else {
    document.write("El segundo es mayor");
}

Bucles (loops)

let numeros = [1, 2, 3];
for (let i = 0; i < numeros.length; i++) {
    document.write(numeros[i] + "
");
}

Ejercicio: Ordenar un array

let v = [9.98, 7.86, 4.53, 8.91, 5.76, 2.31];
for (let i = 0; i < v.length; i++) {
    for (let j = i + 1; j < v.length; j++) {
        if (v[j] < v[i]) {
            let aux = v[i];
            v[i] = v[j];
            v[j] = aux;
        }
    }
}
console.log(v);

Funciones

function calcularMedia(vector) {
    let suma = 0;
    for (let pos in vector) {
        suma += vector[pos];
    }
    return vector.length ? suma / vector.length : 0;
}
let media = calcularMedia([5, 2, 7, 8]); // 5.5

let empleado = {

Apuntes de jQuery

¿Qué es jQuery?

jQuery es una biblioteca de JavaScript que facilita la manipulación de elementos HTML (DOM), la gestión de eventos, las animaciones y las solicitudes AJAX. Su propósito es hacer que la programación en JavaScript sea más sencilla y rápida.

Fue creada para resolver las diferencias entre navegadores y ofrecer una sintaxis más clara y reducida.

Cómo incluir jQuery en un proyecto

Para usar jQuery, basta con incluir el siguiente enlace en la cabecera de nuestro archivo HTML:

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

Ejemplo 1: Hacer desaparecer un párrafo al hacer clic

En este ejemplo, los párrafos desaparecen cuando se hace clic sobre ellos.

<!DOCTYPE html>
<html>
<head>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
<script>
$(document).ready(function(){
  $("p").click(function(){
    $(this).hide();
  });
});
</script>
</head>
<body>

<p>If you click on me, I will disappear.</p>
<p>Click me away!</p>
<p>Click me too!</p>

</body>
</html>
